Resumen

Se detecto conflicto entre dhcp de vlan 609 y dhcp de servicio de cliente , se escalo con zhone quien nos brindo configuración recomendada para configurar dhcp personalizado en ont

Detalle del problema

Se detecto conflicto entre dhcp de vlan 609 y dhcp de servicio de cliente , se escalo con zhone quien nos brindo configuración recomendada para configurar dhcp personalizado en ont

Resolución

Primero tienes que crea un pool nuevo, mira este ejemplo:

1- Add the requeered DHCP lan server:

cpe rg lan dhcp-srvr add LAN-192.168.3.x start-addr 192.168.3.10 end-addr 192.168.3.200 lease-time 86400

Profile "LAN-192.168.3.x" has been created with index 1

confirm:

cpe rg lan dhcp-srvr show 1

cpe rg lan modify 9/7/17 eth 1 ip-addr 192.168.3.1 dhcp-server-profile 1
