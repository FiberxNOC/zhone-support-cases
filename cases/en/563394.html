Case content

Summary

There is intermittent issue with the OLT fx.oldz.juanadiaz.f108, packet loss on the IP of management. Received Zhone commands for equipment stabilization while releasing new firmware.

Notes

The solution is not permanent, Zhone indicates that correction will be made with new software version.

Problem detail

The issue is with the configuration of rule 47.

This rule manages the processing of ARP packets in TLS bridges. ARP packets on this type of bridge should not be directed to the CPU, as they may occupy it and cause packet losses.
