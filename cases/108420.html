Detalle del problema

Se detecto problema al intentar ingresar a las ONT remotamente por http y por telnet, se escalo con zhone, posterior a varias verificaciones nos confirmaron el problema se debía a que las ont de fabrica solo permiten el acceso desde un listado de rangos de direcciones ip por seguridad

Glenn Lee
Type:Solution
Comment:
Hi Jose,

As discussed in the session, for security reason, only the following IP address ranges are allowed management access to the zNID.

192.168.0.0 - 192.168.255.255

10.0.0.0 - 10.255.255.255

172.16.0.0 - 172.31.255.255

Please give it try with any IP within the ranges above and see if the issue is gone.
