Summary

A configuration issue was detected on the 2466GN, as advised by Zhone to leave our ONU in route for it to function correctly.

Problem detail

A technical issue was detected with an ELINE configuration on the 2466GN, as indicated by Zhone. We were instructed to leave our ONT in place for it to function correctly in the scenario provided to Zhone.

Resolution

As we discussed in today's session, there is a need to set the OLT to router mode for the V1-16XC device with serial code 2466GN.
