Problem detail

A blocking was detected at the level of the OLT mxk-F108, where DNS traffic from the FiberX network to an equipment connected to a provisioned ONT on another OLT did not work correctly. Zhone provided us with a configuration to resolve the issue.

Resolution

Here is the translation:

Try to disable the Downstream DNS Filter.

Downstream DNS Filter Feature Enable/Disable discards of downstream DNS queries received on the WAN uplink interface. Default = Enabled.

Via ONT Web UI The option is located on the Config->VLAN Modes screen.
